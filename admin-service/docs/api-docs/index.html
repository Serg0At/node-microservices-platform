Admin Service 1.0.0
  • application/json

Admin dashboard and management microservice for the Arbex platform. Provides dashboard analytics, user management (roles, bans), and proxies article, subscription, promo code, and notification operations to their respective services. gRPC-only, no RabbitMQ — all operations are synchronous. Exposes a gRPC API on port 50055.
