User Service API Reference

Manages public user profiles including display name, username, and avatar. Creates a profile automatically when a user registers. GetProfile is public; UpdateProfile and UploadAvatar require authentication.

All RPCs are exposed through the GraphQL gateway.
GetProfilepublic

Returns the public profile for any user by ID — no authentication required.
